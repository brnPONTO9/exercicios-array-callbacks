# Lista 02 - Métodos de Array e Callbacks

## Objetivo

Praticar o uso de Arrow Functions em conjunto com os principais métodos de manipulação de arrays do JavaScript moderno.

## Conteúdos abordados

* map()
* filter()
* find()
* reduce()
* some()
* every()

## Exercícios

11. Filtrar positivos

12. Nomes em maiúsculo

13. Busca de usuário

14. Cálculo de imposto

15. Contagem de caracteres

16. Filtragem de valores

17. Soma de carrinho

18. Verificação de existência

19. Validação de lista

20. Objeto curto

## Tecnologias

* HTML5
* CSS3
* JavaScript ES6+

## Como executar

1. Abra o arquivo `index.html`.
2. Todas as respostas serão carregadas automaticamente.

## Resultado de aprendizado

Ao concluir esta lista, o aluno estará apto a utilizar callbacks e métodos de arrays amplamente utilizados em projetos ReactJS.
